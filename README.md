\# 🔧 Network Diagnostic Tool



\## 📖 Sobre

Ferramenta simples em Python para testar conectividade de rede utilizando ping.



\## ⚙️ Como executar

1\. Instale Python 3

2\. Execute:

&nbsp;  python ping\_test.py



\## 🎯 Objetivo

Projeto desenvolvido para praticar fundamentos de redes e troubleshooting.
